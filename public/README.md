# AI Chat to Markdown (Gemini & ChatGPT)

A Chrome extension that allows you to export your Google Gemini or ChatGPT history to a Markdown file or copy it to your clipboard.

## Features
- **Multi-Platform**: Works on both [Gemini](https://gemini.google.com) and [ChatGPT](https://chatgpt.com).
- **Save as Markdown**: Downloads the current chat as a formatted `.md` file.
- **Copy to Clipboard**: Copies the chat markdown to your clipboard.

## Installation Instructions

1.  **Download/Locate the Source**: 
    - You should have the folder `d:\VibeProjects\Read Gemini Chat Chrome Extentions` or the `GeminiChatToMarkdown.zip` file.
    - If you have the zip, extract it to a folder.

2.  **Open Chrome Extensions**:
    - Open Google Chrome.
    - Type `chrome://extensions` in the address bar and press Enter.

3.  **Enable Developer Mode**:
    - Toggle the switch for **Developer mode** in the top right corner of the Extensions page.

4.  **Load the Extension**:
    - Click the **Load unpacked** button (top left).
    - Select the folder `d:\VibeProjects\Read Gemini Chat Chrome Extentions` (or where you extracted the files).
    - *Note: Do not select the zip file directly, select the folder containing manifest.json.*

5.  **Pin the Extension**:
    - Click the puzzle piece icon in the Chrome toolbar.
    - Find "Gemini Chat to Markdown" and click the pin icon to keep it visible.

## How to Use
1.  Go to [Gemini](https://gemini.google.com) and open a chat.
2.  Click the extension icon.
3.  Click **Save as Markdown** or **Copy to Clipboard**.

## Troubleshooting
- If you see "No content found", ensure you are on a Gemini chat page and the chat is loaded.
